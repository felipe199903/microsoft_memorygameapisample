Swagger api [location](./config/swagger.json)

## How to use

First, clone the repo.

```bash
$ https://github.com/felipe199903/microsoft_memorygameapisample
```

Then, install the dependencies.

```bash
$ npm install
```

Go to directory.

```bash
$ cd microsoft_memorygameapisample
```

Run.

```bash
$ npm start
```

Open
http://localhost:8000/
